# Downloading and Comparing Brightspace data sets

Use the Javascript file to automate downloading all Brightspace data sets through the browser when logged in with admin account (no need for OAuth 2.0 or access tokens or to download files individually using Data Hub). Once .zip files are saved to disk, use Python files to count the rows for each data set file and/or to compare two sets of data set files with the same names in different folders, such as EDW and DHUB. The differences between the files will be output to a third folder, with summary data of row counts and if a diff file was created for that data set.
